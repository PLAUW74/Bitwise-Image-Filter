# Bitwise Image Filter (C++)

A small C++ project that uses bit manipulation to apply a color inversion filter to an image.

---

## How to Use

1. Place your input image (`image.jpg`) in the same folder as `main.cpp`.
2. Compile the program:
```bash
g++ main.cpp -o bitfilter
```
3. Run the program:
```
./bitfilter
```
